# PS_l-1
## "Разработка телефонной книги с авторизацией"

### 1. Цель работы
Разработать оконное приложение "Телефонная книга" с формой авторизации.
Провести тестирование программы продемонстрировать возможный функционал.

### 2. Структура проекта

- main.py # Главный файл запуска
- config.py # Настройки и конфигурация
- auth.py # Модуль авторизации
- phonebook.py # Модуль телефонной книги
- users.json # Файл с пользователями
- contacts.json # Файл с контактами


### 3. Описание модулей

#### 3.1 config.py
Содержит все настройки приложения:
- Настройки шрифтов (FontConfig)
- Настройки размеров (SizeConfig)
- Настройки цветов (ColorConfig)
- Настройки файлов (FileConfig)
- Функция центрирования окон

#### 3.2 auth.py
Реализует функционал авторизации:
- Класс AuthManager для управления пользователями
- Загрузка/сохранение пользователей в JSON
- Проверка учетных данных
- Добавление новых пользователей
- Смена пароля

#### 3.3 phonebook.py
Содержит логику работы с контактами:
- Класс ContactManager для управления контактами
- Класс ContactCard для отображения контакта в виде карточки
- Класс PhoneBookUI для создания интерфейса
- CRUD операции с контактами
- Поиск контактов

#### 3.4 main.py
Главный файл приложения:
- Класс LoginWindow - окно авторизации
- Класс PhoneBookApp - главное окно телефонной книги
- Обработка событий и связь между модулями

### 4. Функциональные возможности

1. **Авторизация пользователей**
   - Вход по имени и паролю
   - Защита от неверного ввода
   - Сообщения об ошибках

2. **Управление контактами**
   - Просмотр всех контактов в виде карточек
   - Добавление нового контакта
   - Редактирование существующего (кнопка "Коррекция")
   - Удаление контакта
   - Поиск по имени

3. **Сохранение данных**
   - Автоматическое сохранение в JSON файлы
   - Загрузка при запуске
